Надежный стальной портал

Полностью автоматическая и самостоятельно перемещающаяся моечная установка для автомобилей промышленного назначения MAGNUM оснащена прочно сваренным стальным порталом с полным цинкованием огнем для длительного срока эксплуатации.

Оптимальное давление прижима щеток

Электронное управление моющими валиками (ПЛК) обеспечивает бережное и эффективное мытье с учетом контуров автомобиля.

Ходовой механизм с низким полом для полного покрытия автомобиля

В стандартной комплектации все моечные установки для автомобилей промышленного назначения линейки MAGNUM оснащены нашим ходовым механизмом с низким полом. Это значит, что самостоятельно опускаемые автобусы с низким полом можно полностью мыть боковыми валиками. Расстояние между нижней кромкой вращающегося бокового валика и верней кромкой пола составляет всего 175 мм.

Модульная система

Благодаря модульной системе, MAGNUM можно дополнять подходящей машинной оснасткой для любой сферы применения автомойки. Моечная установка для автомобилей промышленного назначения, оснащенная в стандартной комплектации 3 валиками, может по желанию также быть реализована в версии с двумя валиками (исполнение без верхнего валика).
